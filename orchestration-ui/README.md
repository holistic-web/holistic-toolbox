# Orchestration UI

User interface for the orchestration system. It is hosted with firebase and written in VueJS

It's hosted at: https://holistic-home-5134d.firebaseapp.com

## Setup
1. install dependencies
	```
	> npm install
	```

## Running
1. just run the app :)
	```
	> npm run serve
	```

## Deploying
1. build the project
	```
	> npm run build
	```

2. deploy the project
	```
	> firebase deploy
	```
